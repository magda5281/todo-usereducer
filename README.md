# 🚀 Basic Todo App with `useReducer` (React + TypeScript + Vite)

This is a simple **To-Do App** built using **React**, **TypeScript**, and **Vite**, showcasing the use of the `useReducer` hook for state management.

---

## 📌 Features
- ✅ Add, toggle, and delete todos
- ✅ Uses `useReducer` for centralized state management
- ✅ Dark mode support (auto-detects system preference)
- ✅ Fast and optimized with Vite
